# Botim OP Docs

Documentation platform for Botim Open Platform built with Nuxt 4.

## Tech Stack

- [Nuxt 4](https://nuxt.com) — Full-stack Vue framework
- [Nuxt UI v4](https://ui.nuxt.com) — UI components
- [MDC](https://mdc.nuxt.com) — Markdown Components
- [Tailwind CSS 4](https://tailwindcss.com) — Styling
- [Monaco Editor](https://microsoft.github.io/monaco-editor/) — Markdown editor

## Prerequisites

- Node.js 22 LTS
- Corepack enabled (`corepack enable`) — automatically provides Yarn 4

## Setup

```bash
yarn install
```

## Development

```bash
yarn dev
```

## Build & Run

```bash
yarn build
yarn start
```

# API Endpoints

This page documents all available REST API endpoints for the Botim Open Platform.

## Base URL

| Environment | Base URL |
|-------------|----------|
| Sandbox | `https://sandbox-api.botim-op.com/v3` |
| Production | `https://api.botim-op.com/v3` |

## Authentication

All API requests require authentication via an API key in the `Authorization` header:

```bash
Authorization: Bearer YOUR_API_KEY
```

## Messages

### Send Message

Send a message to a user.

```
POST /messages
```

**Request Body:**

```json
{
  "to": "user_12345",
  "type": "text",
  "content": {
    "body": "Hello, World!"
  }
}
```

**Response (200):**

```json
{
  "id": "msg_abc123",
  "status": "sent",
  "timestamp": "2025-01-15T10:30:00Z"
}
```

### Get Message Status

Retrieve the delivery status of a message.

```
GET /messages/:id
```

**Response (200):**

```json
{
  "id": "msg_abc123",
  "status": "delivered",
  "sentAt": "2025-01-15T10:30:00Z",
  "deliveredAt": "2025-01-15T10:30:02Z"
}
```

### List Messages

Retrieve a paginated list of messages.

```
GET /messages?limit=20&cursor=msg_abc123
```

**Query Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `limit` | number | 20 | Items per page (max 100) |
| `cursor` | string | — | Cursor for pagination |
| `direction` | string | `desc` | Sort direction |

## Users

### Get User Profile

```
GET /users/:id
```

**Response (200):**

```json
{
  "id": "user_12345",
  "displayName": "John Doe",
  "avatar": "https://cdn.botim.com/avatars/user_12345.jpg",
  "status": "active",
  "lastSeen": "2025-01-15T10:00:00Z"
}
```

### Search Users

```
GET /users/search?q=john&limit=10
```

## Payments

### Create Payment Request

```
POST /payments
```

**Request Body:**

```json
{
  "to": "user_12345",
  "amount": {
    "value": 29.99,
    "currency": "USD"
  },
  "description": "Premium subscription",
  "callbackUrl": "https://your-app.com/api/payment-callback"
}
```

**Response (201):**

```json
{
  "id": "pay_xyz789",
  "status": "pending",
  "paymentUrl": "https://pay.botim.com/pay_xyz789",
  "expiresAt": "2025-01-15T11:30:00Z"
}
```

### Get Payment Status

```
GET /payments/:id
```

## Webhooks

### Register Webhook

```
POST /webhooks
```

**Request Body:**

```json
{
  "url": "https://your-app.com/api/webhooks/botim",
  "events": ["message.received", "payment.completed"],
  "secret": "your_webhook_secret"
}
```

### List Webhooks

```
GET /webhooks
```

### Delete Webhook

```
DELETE /webhooks/:id
```

## Error Codes

All errors follow a consistent format:

```json
{
  "error": {
    "code": "INVALID_REQUEST",
    "message": "The 'to' field is required",
    "details": {
      "field": "to",
      "reason": "missing"
    }
  }
}
```

| Code | HTTP Status | Description |
|------|-------------|-------------|
| `INVALID_REQUEST` | 400 | Request body is malformed or missing required fields |
| `UNAUTHORIZED` | 401 | Invalid or missing API key |
| `FORBIDDEN` | 403 | Insufficient permissions |
| `NOT_FOUND` | 404 | Resource not found |
| `RATE_LIMITED` | 429 | Too many requests |
| `INTERNAL_ERROR` | 500 | Internal server error |

## Rate Limits

| Plan | Requests/minute | Messages/day |
|------|----------------|--------------|
| Free | 100 | 1,000 |
| Developer | 1,000 | 50,000 |
| Business | 10,000 | Unlimited |
| Enterprise | Custom | Custom |

Rate limit headers are included in every response:

```
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1705312200
```
